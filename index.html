Features

UIkit gives a comprehensive collection of HTML, CSS and JS components
which is simple to use, easy to customize and extendable

LESS

UIkit is developed in LESS and SASS to write well-structured, extendable code which is easy to maintain.

Components

A collection of small, responsive components using consistent and conflict-free naming conventions.

Customizer

UIkit's very basic style can be extended with themes and is easy to create your own look.

Responsive

With the mobile-first approach UIkit provide a consistent experience from phones and tablets to desktop.
